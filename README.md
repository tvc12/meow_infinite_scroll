## Getting Started

+ Clone this repo with command line

```sh
git clone https://github.com/tvc12/meow_infinite_scroll.git && cd meow_infinite_scroll && yarn
```

+ Run project

```sh
yarn serve --open
```

Open browser in http://localhost:8080 and test it
